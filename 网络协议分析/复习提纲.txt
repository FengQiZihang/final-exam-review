第 1 章 网络协议概述
核心知识点
1.	网络协议的定义、三要素（语法、语义、时序）及两种形态
2.	分层网络体系结构的设计思想、优势与不足
3.	OSI/RM 参考模型七层结构及各层核心功能
4.	TCP/IP 参考模型四层结构及各层核心功能、协议之间的依赖关系
第 3 章 数据链路层协议
核心知识点
1.	数据链路层核心功能（成帧、差错检测、可靠交付、媒体访问、流量控制）
2.	PPP 协议的组成（帧格式、LCP、NCP、认证协议）
3.	PPP 帧格式各字段含义及透明传输实现（字节填充、零比特填充）
4.	LCP 协议的功能、报文种类
5.	NCP 协议的功能，IPCP 协议的核心作用
6.	PAP 与 CHAP 认证协议的流程、报文格式及安全性对比
7.	以太网协议（DIX Ethernet V2、IEEE 802.3）帧格式及各字段含义
8.	MAC 地址的结构、表示方法及分类（单播、多播、广播）
第 4 章 IP 层协议
核心知识点
1.	IP协议的核心作用、服务特性（尽力而为）
2.	IPv4 地址分类（A/B/C/D/E 类）、特殊地址（网络地址、广播地址、环回地址、私网地址）
3.	子网划分的原理、子网掩码的作用及计算方法
4.	CIDR 技术（无分类域间路由）的特点、斜线记法及路由聚合
5.	ARP 协议的作用、工作原理（广播请求、单播响应）、缓存机制
6.	IPv4分组格式（固定首部字段、可选字段）及校验和计算方法
7.	IP分组分片与重组的触发条件及实现机制
8.	ICMP 协议的作用、报文类型（差错报文、信息报文）及应用（ping、traceroute）
第5章 路由协议
核心知识点
1.	路由算法分类（静态 / 动态、距离向量 / 链路状态 / 路径向量）
2.	RIP 协议的特点、工作原理（距离向量算法、跳数度量）、报文格式
3.	RIP 协议的慢收敛问题及改进措施（水平分割、毒性逆转、触发更新）
4.	RIPv1 与 RIPv2 的区别（子网掩码、认证、组播）
5.	OSPF 协议的特点、工作原理（链路状态算法、SPF 算法）
6.	OSPF 的区域划分（主干区域、非主干区域）及路由器类型（IR、ABR、BR、ASBR）
7.	OSPF 协议的五种报文类型、DR、BDR
8.	BGP 协议的特点、作用（AS 间路由）、工作原理（TCP 连接、增量更新）
9.	BGP 协议的四种报文类型及状态机
第 6 章 传输层协议
核心知识点
1.	传输层的核心作用（进程间逻辑通信、弥补网络层服务缺陷）
2.	端口的定义、分类（熟知端口、临时端口）及作用
3.	多路复用与多路分解的实现原理
4.	UDP 协议的特点、报文格式及校验和计算
5.	TCP 协议的特点（面向连接、可靠传输、全双工、流量控制、拥塞控制）
6.	TCP 报文段格式（固定首部字段、标志位、窗口字段等）
7.	TCP 连接管理（三次握手建立连接、四次挥手释放连接）
8.	TCP 可靠传输机制（序号、确认、超时重传、累计确认）
9.	TCP 流量控制（滑动窗口机制）
10.	TCP 拥塞控制（慢启动、拥塞避免、快重传、快恢复）
第 7 章 应用层协议
核心知识点
1.	DNS 协议的功能、域名空间结构（分层）、域名服务器类型（根、顶级、权限、本地）
2.	DNS 解析过程（迭代查询、递归查询）及报文格式
3.	DHCP 协议的功能、工作流程（发现、提供、选择、确认）及报文种类
4.	DHCP 地址分配方式（静态、动态）及租用期管理
5.	FTP 协议的功能、C/S 架构、双连接（控制连接、数据连接）机制
6.	电子邮件系统组成（用户代理、邮件服务器）及核心协议（SMTP、POP3）
7.	HTTP 协议的功能、URL 格式、请求 / 响应报文结构
8.	HTTP 方法（GET、POST、HEAD 等）
